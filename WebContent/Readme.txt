<< 실행 및 테스트 방법 >>

테스트 id : dongduk / pw: 1234

- 메인 화면(movie) : 상영중인 영화 목록을 볼 수 있고, "상세정보" 버튼을 누를 시, 
  해당 영화의 상세 정보를 확인할 수 있으며, "예매하기" 버튼을 누를 경우, 바로 선택한 영화의 예매 화면으로 넘어간다.
	
- 회원 가입 : 사용자는 아이디, 비밀번호, 이름, 이메일, 전화번호 정보 입력을 한 후, 회원가입을 한다. 
  회원가입과 취소 버튼을 누를 경우 모두 로그인 창으로 이동한다.
	
- 로그인 : 사용자의 아이디와 비밀번호를 이용하여 로그인에 성공하면 메인화면(movie)으로  넘어간다. 
  실패할 시, 오류를 띄워준다. 회원가입 버튼을 누를 경우, 회원가입으로 넘어간다.
	
- 회원정보 : 상단바의 '마이페이지'에 있는 '회원정보' 클릭 시, 회원정보 확인 창으로 넘어가 회원 정보를 확인할 수 있다.
  수정 버튼을 누를 경우, 회원정보 수정 페이지로 넘어가고, 취소 버튼을 누를 경우, 메인화면(movie)로 넘어가며, 
  회원탈퇴 버튼을 누를 시, 회원이 탈퇴되어 로그아웃되고 로그인 창으로 돌아간다.
	
- 회원정보 수정 : 아이디를 제외한 나머지 정보들을 수정할 수 있다. 취소 버튼을 누를 경우, 정보가 수정이 되지 않는다.
	
- 영화 상세정보 : 선택한 영화에 대한 상세정보와 함께 영화에 대한 리뷰를 남길 수 있다. 
  (단, 사용자 본인이 쓴 댓글만 수정과 삭제가 가능하다.) 
  추가, 수정 혹은 삭제를 하면 결과를 확인할 수 있도록 해당영화 상세정보페이지로 이동한다.

- 영화 예매 : 영화 예매는 상단바의 '예매'에서 '상영 시간표'에서 사용자가 원하는 영화와 시간대를 선택하거나 
  메인 화면에 있는 예매하기 버튼 또는 상단바의 '예매'에서 '예매하기'를 통해 영화 예매를 진행할 수 있다. 
  영화와 시간대 선택 후, 좌석을 선택하게 되면 포인트 룰렛을 통해 포인트 적립이 되고, 예매 완료 창이 뜬다. 
  (영화는 전날 예매만 가능하며, 좌석은 다중 선택 가능하고, 이미 선택된 좌석은 선택할 수 없다. 또한, 좌석 선택안하면 경고창이 뜬다.)
  포인트 룰렛의 경우 당첨된 포인트 곱하기 선택한 좌석수만큼 포인트 적립이된다.
	
- 영화 검색 : 상단바의 '검색' 기능을 통해 현재 상영중인 영화 제목을 검색하여 정보를 찾을 수 있다. 
  (예: '보' 검색시, '보'가 들어가는 '보헤미안랩소디', '담보' 모두 검색된다.)
	
- 예매 확인 :  예매가 끝나면 예매 완료 창이 뜨고, 예매 내역 확인하기 버튼 클릭 시, 
  사용자가 예매한 티켓 내역을 확인할 수 있으며, 상단바의 마이페이지에 있는 예매확인을 통해서도 확인이 가능하다. 
  또한, 티켓 각각 예매 취소가 가능하다. (예매 취소 시, 포인트도 자동 차감된다.)
	
- 포인트 확인 :  포인트 룰렛을 통해 적립된 포인트는 상단바의 마이페이지에 있는 포인트 확인을 통해 누적 포인트를 확인할 수 있다. 

<<주의사항>>
1. 회원정보 확인 및 수정, 영화 예매, 예매 및 포인트 확인의 경우 모두 로그인을 해야 가능합니다. 
  (로그인이 안되어있을 경우, 자동으로 로그인화면으로 넘어갑니다.)
2. 좌석 선택 화면의 경우 외부 css를 가져와 썼기때문에 의자 그림이 안나올 수 있습니다. 하지만 선택은 잘됩니다.
